# Materials List

This document lists all the materials needed per student station to complete the Arduino LED & Button workshop. Make sure each station is fully set up before students arrive.

---

## Per Student Station

| # | Item | Quantity | Notes |
|---|------|----------|-------|
| 1 | Arduino Nano | 1 | Any version works (Nano v3 recommended) |
| 2 | Push Button | 1 | Standard 4-leg tactile push button |
| 3 | LED | 1 | Any color |
| 4 | Breadboard | 1 | Mini breadboard recommended for classroom use |
| 5 | Jumper Wire Cables | 5 | Male-to-male recommended for breadboard use |
| 6 | Computer | 1 | Must have a USB port available |
| 7 | USB Cable (Micro-USB or Mini-USB) | 1 | Match to your Arduino Nano's port type |

---

## Software Requirements

Before the workshop, make sure the computer at each station has the following installed:

- **Arduino IDE** — [Download here](https://www.arduino.cc/en/software)
- **Arduino Nano USB Driver** (CH340) — may need to be installed manually on Windows. [Download here](https://sparks.gogo.co.nz/ch340.html)

---

## Notes for Staff

- Test each Arduino Nano ahead of time by uploading a simple blink sketch to confirm it works.
- Check that each USB cable supports **data transfer**, not just charging — some cables are charge-only and will not work.
- LEDs are directional — the **longer leg is positive (+)**. If an LED doesn't light up, try flipping it.
- Have a few extra components on hand in case something is damaged or lost during the workshop.
